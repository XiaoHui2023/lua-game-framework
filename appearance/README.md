# appearance

外观修正模块，负责把外观数据、修正器和渲染动作分开。

- modifier 描述外观如何变化。
- renderer 负责把最终外观应用到宿主对象。
- 模块保持引擎无关，具体表现由运行时或上层接入。

## 设计特性

### 修正分层

外观变化先以修正器表达，再统一渲染。皮肤、颜色、缩放等表现可以按职责拆开。

### 渲染边界

框架层不直接操作引擎 API。渲染器只接收整理后的外观结果，具体句柄操作留给接入层。

## 目录

```text
init.lua      # 外观模块门面
modifier.lua  # 外观修正器
renderer.lua  # 外观渲染入口
test.lua      # 外观行为测试
```
