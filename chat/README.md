# chat

聊天模块，处理玩家输入、消息入库和消息变更广播。

- 输入先进入 `ON_INPUT`，默认实现再写入消息历史。
- 历史列表有最大数量限制。
- 消息变化通过 callback API 通知 UI 或其他系统。

## 设计特性

### 输入归一

玩家输入先被整理成统一消息请求。过滤、频道切换或命令解析可以注册在输入 API 上。

### 历史列表

消息写入后进入列表并按上限裁剪。读取方拿到的是当前历史列表，适合 HUD 聊天窗口直接监听刷新。

## 目录

```text
apis.lua     # 聊天 callback API
impl.lua     # 默认输入与消息处理
init.lua     # 聊天门面与历史列表
settings.lua # 消息上限等默认参数
```
