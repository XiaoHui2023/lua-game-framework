# sound

声音模块，封装声音对象、默认参数和声音相关 runtime API。

- 声音对象表达一次或持续声音。
- settings 放默认音量、距离等参数。
- apis 声明播放、停止和宿主能力。

## 设计特性

### 声音对象

声音以对象管理，便于保存句柄、目标和释放动作。UI 音效、世界音效和循环音都可以复用同一模型。

### 抽象播放

框架发出播放请求，运行时完成实际引擎调用。模块不直接保存 Y3 句柄创建细节。

## 目录

```text
apis.lua     # 声音 callback API
init.lua     # 模块门面
object.lua   # 声音对象
settings.lua # 声音默认参数
```
