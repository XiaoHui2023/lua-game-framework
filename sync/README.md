# sync

同步模块，为异步环境提供带唯一头的消息通道。

- 每次 `register` 创建独立同步通道。
- 发送前触发本地 on_send，收到宿主消息后触发 on_receive。
- 底层发送和监听由 runtime 注册 callback handler。

## 设计特性

### 独立通道

模块为每个同步对象生成唯一 head。多个系统可以各自注册同步通道，不需要手动维护消息前缀。

### 发送接收分离

同步对象可调用发送，也可注册接收监听。框架只处理通道语义，具体网络或引擎同步由运行时接入。

## 目录

```text
apis.lua # 同步 callback API
```
