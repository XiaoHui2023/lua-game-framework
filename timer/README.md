# timer

计时器模块，为框架提供循环、延迟和 reactive 调度驱动。

- `loop` 创建循环任务并返回取消函数。
- `delay` 创建延迟任务并返回取消函数。
- `driver` 可供 reactive 使用同一套时间推进。

## 设计特性

### 中心计时

模块维护累计运行时间，循环和延迟都通过统一核心处理。上层不需要直接面对宿主计时器。

### 可取消任务

计时任务创建后立即返回取消函数。对象销毁时可以把取消函数挂入生命周期，避免计时器悬挂。

## 目录
