# visual_effect

视觉特效模块，声明特效创建参数和运行时特效能力。

- 特效目标可以是世界坐标或单位句柄。
- options 覆盖插槽、角度、缩放、持续时间和跟随规则。
- settings 放默认特效参数。

## 设计特性

### 目标统一

特效既可以挂到点，也可以挂到单位目标。调用方传入同一套 options，运行时决定具体引擎创建方式。

### 生命周期参数

持续时间、分离、跟随旋转和跟随缩放都放在 options。技能、状态和场景表现可以用同一套字段描述特效生命周期。

## 目录

```text
apis.lua     # 特效 callback API
init.lua     # 特效类型与门面
settings.lua # 特效默认参数
```
